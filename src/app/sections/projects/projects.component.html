A brief introduction to my best work projects

Automotive EDI

Microsoft BizTalk Server 2016Microsoft Dynamics AX.NETJavaSQL Server 2014Angular 11

For a large player in the automotive sector we have created a system for managing and exchanging EDI documents with customers and suppliers.

The system currently manages more than 200 partners, with an average of 150,000 documents per month and an SLA of 99.95%. My role in the project was that of architect, in particular I helped to size the BizTalk infrastructure and set up the solution for development. I also helped develop the OFTP document exchange connector and document tracking architecture myself, with a web dashboard to show users the status of various sent and received documents.

Document System

Microsoft SharePoint 2013Microsoft Dynamics NAV.NET

For a customer who works in the production and sale of medical gases we have created a document archiving system by combining SharePoint 2013 and Microsoft Dynamics NAV. The system currently manages around 5 million documents a year including invoices, shipping notes, intervention reports and other documents. My role in the project was that of architect and developer of the SharePoint part, including the integration part with Dynamics NAV and a certified external provider for replacement archiving.

Patient Portal

Microsoft SharePoint 2010Microsoft Dynamics NAV.NETAngular 11

For a customer who works in the production and sale of medical gases, we have created a portal that allows doctors, pharmacists, hospitals and local health companies to enter requests for medical gases for their patients. The system was initially developed in 2011 with SharePoint 2010 and Microsoft Dynamics NAV. In 2021 the system has been completely revised for the web part which has been rewritten in .NET Core as regards the APIs and in Angular as regards the web client.

My role, in the initial project, was that of architect and developer of the web part created in SharePoint. Later, in the revised project I helped set up the backend architecture, especially the development of the REST API of the system and the integration with Dynamics NAV which was heavily revised.

Pension fund management

Microsoft Dynamics NAV.NETAngular 12

I worked on the refactoring of a legacy application dedicated to pension fund management. The original application was made in Asp.Net Web forms, integrated with Microsoft Dynamics NAV ERP via web services; the application was offered to several customers as a service and a customized deployment was created for each customer on which specific customizations were made.

The new application was created with .NET Core and Angular 12 and follows a SAAS (Software As A Service) approach, with a single deployment that allows you to manage multiple customers simultaneously, each with its own customizations and plugins. The new approach has drastically reduced startup times for a new customer and software maintenance costs as there is now a single codebase to maintain.

My role in this project was that of Architect of both the backend and the frontend part. I also contributed to the development of the main components of all solution.

Compressor Design

.NETMatlabKnockOut.jsSQL Server 2014

For a customer, world leader in the production of compressors and air separation units, we have created a web application to support the design and simulation phases of compressors. The application collects the main data on the type of compressor to be built and the gas mixture to be used and provides a rough dimensioning of the various processor components. It also allows you to simulate the operation of a compressor designed by the system under certain conditions.

The application was built with ASP.NET MVC, with intensive use of the KnockOut.js library to build the most important components. It also integrates with two external components, a Matlab engine capable of performing engineering calculations and a library dedicated to thermodynamic calculations on gas mixtures.

My role in this project was that of Architect of the .NET solution. I also personally contributed to the development of some components.
